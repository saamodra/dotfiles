# Requirements


## Fonts

Install wps symbol fonts

```
git clone https://github.com/IamDH4/ttf-wps-fonts.git
cd ttf-wps-fonts
sudo bash install.sh
```

Install fonts-open-sans

```
sudo apt-get update
sudo apt install fonts-open-sans
```
